# Shree Kashtbhanjan Travels - Web App

This directory contains the Next.js web application for **Shree Kashtbhanjan Travels**.

production Url Live  : https://kashtbhanjan-travels.vercel.app/

For the full project overview, features, styling design, and responsive optimizations, please refer to the main **[Root README](../README.md)**.

## 🚀 Quick Start

Ensure you have [Node.js](https://nodejs.org/) installed, then run the following commands in this directory:

### 1. Install Dependencies
```bash
npm install
```

### 2. Start the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see the site running.

### 3. Build for Production
```bash
npm run build
```

### 4. Run the Production Server
```bash
npm run start
```

### 5. Run Linter
```bash
npm run lint
```
